**the title of project:Footballer robots**
**developer:Vahid AtashBar and HamidReza Kordi**
**The project included two robots.
  A powerful robot that was in charge of Hamid and a defense robot that was in charge of Vahid.**
**This code tells what each bot should do to make a good team.**
**A class named PID is defined. It is called in every code for the robots.
 The defense robot moves the ball away. The attack robot follows the position of the ball.**
**Using PID controller
  Discretization using time sampling at alternate times and periodically.**
